# Tic-Tac-Toe Game

A simple and interactive Tic-Tac-Toe game built using **HTML**, **CSS**, and **JavaScript**. This project is designed for beginners to understand the basics of web development and JavaScript game logic.

## Features

- Two-player gameplay (Player X and Player O)
- Winner detection
- Draw detection
- Restart game button
- Responsive and user-friendly interface
- HTML, CSS, and JavaScript combined in a single file

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Project Structure

```
tic-tac-toe/
│── index.html
└── README.md
```

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` or open it in any web browser.

## Game Rules

1. Player **X** starts the game.
2. Players take turns placing their symbols on the board.
3. The first player to align three symbols horizontally, vertically, or diagonally wins.
4. If all cells are filled and no player wins, the game ends in a draw.
